Project Goal:
Green Homeland is an interactive website designed to raise environmental awareness and teach sustainable living habits. Users can access information about recycling, energy conservation, natural resource protection, and eco-friendly practices.

Target Audience:

Students

Anyone interested in environmental awareness

Educational institutions and environmental projects

Key Features:

Educational content: Simple, visual, and easy-to-understand information

Interactive guides: Users can engage and learn actively

Mobile-friendly design: Accessible on all devices

Clean interface: Easy navigation for all users

Technologies Used:
HTML, CSS, JavaScript

Project Outcomes / Skills Gained:

Web development experience

UI/UX design skills

Creating interactive educational content

Increased understanding of sustainability and environmental issues
